Detailed Description

Contains the functions to determine if a cell is alive or dead in a matrix.

Can create, initialize, and free a frame structure, which has a 2d matrix as well as the lengths of the rows and columns. Also has functions to determine if cells in an hedge, torus, or klein pattern are alive or dead.
